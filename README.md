# cultura_paz_VENUS

Este proyecto aborda la violencia emocional en parejas adolescentes dentro de la comunidad escolar. A través de un cartel preventivo y recursos informativos, se promueve el autocuidado y la cultura de paz. Las señales de alerta, acciones de autocuidado y rutas de apoyo están claramente presentadas.
